## 🚀 public-mqtt-client-core

`mqtt-core-java`는 두 가지 핵심 작동 모드를 지원하는 MQTT Java 라이브러리입니다.

### 1. Normal Mode (프로토콜 변환)
* **Protocol Conversion (MQTT <-> RESTful API)**
* MQTT 클라이언트의 메시지를 RESTful API 호출과 같은 다른 프로토콜로 변환하는 기능을 지원합니다.

### 2. Bridge Mode (메시지 패싱)
* **Inter-Cloud Broker Bridging**
* 서로 다른 두 클라우드 환경에 위치한 MQTT 브로커 간의 메시지를 안정적으로 중계(Passing)하고 전달합니다.
